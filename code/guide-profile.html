Хөтөчийн сэтгэгдэл

Маш найрсаг, мэргэжлийн хөтөч байсан.

Аяллын үеэр бүх мэдээллийг ойлгомжтой тайлбарлаж өгсөн. Замын аюулгүй байдлыг сайн анхаарч, асуусан зүйл бүрт шууд хариулж байсан.
